What style of circuit building UI?

How should components be placed and connected?

BREADBOARD
Components snap into holes · Wires connect rows

A — Breadboard

Physical breadboard with hole grid. Drag components in, connect with virtual wires. Matches how hobbyists actually prototype pedals.

B — Schematic

Traditional circuit diagram with standard symbols (zigzag resistors, parallel-line caps, triangle op-amps). Wire nodes by clicking. Engineering-style.

C — Node Graph

Components as draggable nodes with input/output ports. Connect by drawing wires between ports. Like a DAW's modular routing, but for circuits.
